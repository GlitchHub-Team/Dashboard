# Dashboard

repository MVP per lo sviluppo della Dashboard

# Setup
1. Clona il repository Infrastructure con `git clone https://github.com/GlitchHub-Team/Infrastructure.git`
2. Crea il file .env nella cartella Infrastructure e prendi il contenuto del file .env in Bitwarden dalla cartella *Infrastructure*
3. Crea i file gateway1.creds e gateway2.creds nella cartella Infrastructure/testCreds e prendi il contenuto dei file gateway1.creds e gateway2.creds in Bitwarden dalla cartella *Infrastructure*
4. Crea il file stream_setupper.creds nella cartella Infrastructure/natsManager e prendi il contenuto del file stream_setupper.creds in Bitwarden dalla cartella *Infrastructure*
5. Crea il file server.key nella cartella Infrastructure/nats/certs e prendi il contenuto del file server.key in Bitwarden dalla cartella *Infrastructure*
6. Crea il file admin_test.creds nella cartella backend e prendi il contenuto del file admin_test.creds in Bitwarden dalla cartella *Global*
7. Crea il file dashboard.creds nella cartella backend e prendi il contenuto del file dashboard.creds in Bitwarden dalla cartella *Dashboard*
